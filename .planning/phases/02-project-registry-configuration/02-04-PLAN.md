1. **Add two Tauri commands** to `src-tauri/src/commands/config_commands.rs`:

   - `export_config(app_handle: AppHandle, path: String) -> Result<(), ConfigError>`:
     Load config via load_or_create_config, serialize to pretty JSON with serde_json::to_string_pretty, write to the given `path` using std::fs::write.

   - `import_config(app_handle: AppHandle, path: String) -> Result<AppConfig, ConfigError>`:
     Read file at `path` with std::fs::read_to_string, deserialize as AppConfig via serde_json::from_str, validate that version field is present (it will be since it's required by the struct), save to app config path via save_config, return the imported config.

   Register both new commands in `src-tauri/src/lib.rs` invoke_handler (add to the existing list of commands).

2. **Create `src-ui/src/pages/Settings.tsx`**:

   **Header:**
   - "Settings" heading: text-xl (20px) font-semibold text-gray-50

   **Section: General** (Card with p-4, mt-4):
   - **Refresh interval**: Label "Refresh interval" + Input (type="number", min=5, max=300) + "seconds" span
     - Label style: text-xs uppercase tracking-wider text-gray-400
     - Helper text below: "How often to check git status, in seconds." -- text-xs text-gray-500
     - Default value from store: `settings.refresh_interval` (30)
     - Auto-save on blur: call `store.updateSettings({ refresh_interval: parseInt(value) })`
   - **Start minimized**: Label "Start minimized to tray" + checkbox input
     - Use a styled checkbox (`<input type="checkbox" />` with Tailwind classes)
     - On change: immediately call `store.updateSettings({ start_minimized: checked })`
   - **Start with Windows**: Label "Launch when Windows starts" + checkbox
     - On change: immediately call `store.updateSettings({ start_with_windows: checked })`

   **Section: Theme** (Card with p-4, mt-6):
   - Text: "Dark mode" with a note: "Light theme coming in a future version." -- text-xs text-gray-500
   - No interactive control (dark-only for v1 per user decision)

   **Section: Data** (Card with p-4, mt-6):
   - Section heading: "Configuration"
   - **Export button**: "Export Config" -- Button variant="outline", on click:
     ```typescript
     import { save, open } from '@tauri-apps/plugin-dialog';
     import { invoke } from '@tauri-apps/api/core';

     const handleExport = async () => {
       const filePath = await save({
         defaultPath: 'grove-config.json',
         filters: [{ name: 'JSON', extensions: ['json'] }],
       });
       if (filePath) {
         await invoke('export_config', { path: filePath });
       }
     };
     ```
   - **Import button**: "Import Config" -- Button variant="outline", on click:
     ```typescript
     const handleImport = async () => {
       const selected = await open({
         multiple: false,
         filters: [{ name: 'JSON', extensions: ['json'] }],
       });
       if (selected) {
         await invoke('import_config', { path: selected as string });
         await store.loadConfig(); // reload after import
       }
     };
     ```
   - Helper text: "Export saves your projects and settings. Import replaces current configuration."
