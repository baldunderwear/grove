  <action>
    Create `src-tauri/src/git/` module with four files:

    **git/error.rs** - GitError enum following the ConfigError pattern:
    - Variants: Git(git2::Error), Io(std::io::Error), RepoNotFound(String), BranchNotFound(String), UnexpectedConflict(Vec<String>), MergeAborted(String)
    - Implement `serde::Serialize` by serializing as `self.to_string()` (same pattern as ConfigError)
    - Use `#[derive(Debug, thiserror::Error)]`

    **git/branches.rs** - Worktree branch listing:
    - `BranchInfo` struct (derive Serialize, Clone): name (String), ahead (usize), behind (usize), last_commit_message (String), last_commit_timestamp (i64), is_dirty (bool), worktree_path (String)
    - `WorktreeInfo` struct (internal): name (String), branch (String), path (String)
    - `pub fn list_worktree_branches(project_path: &str, branch_prefix: &str, merge_target: &str) -> Result<Vec<BranchInfo>, GitError>`:
      1. Open main repo with `Repository::open(project_path)`
      2. Get worktree names with `repo.worktrees()`
      3. For each worktree name: `repo.find_worktree(name)` to get path, then `Repository::open(wt_path)` to open as separate repo
      4. Get HEAD of worktree repo: `wt_repo.head()?.shorthand()` for branch name
      5. Filter by branch_prefix (skip if branch doesn't start with prefix)
      6. Get ahead/behind: find branch OID and merge_target OID from the MAIN repo (not worktree repo), use `repo.graph_ahead_behind(branch_oid, target_oid)`
      7. Get last commit: `repo.find_commit(branch_oid)` for message and time
      8. Check dirty: call `is_worktree_dirty` on the worktree path
      9. Also include the main repo's own HEAD branch if it matches prefix (worktrees() doesn't list the main working tree)
    - IMPORTANT PITFALL: Branches exist in the main repo's refs. Use the main repo to look up OIDs via `repo.find_branch(branch_name, BranchType::Local)`, but open each worktree path as a separate Repository to check dirty status.
    - IMPORTANT PITFALL: Windows paths -- normalize with `replace('\\', "/")` before comparing paths.

    **git/status.rs** - Dirty status check:
    - `pub fn is_worktree_dirty(worktree_path: &str) -> Result<bool, GitError>`:
      1. Open repo at worktree_path
      2. Create `StatusOptions::new()` with `include_untracked(true)` and `recurse_untracked_dirs(false)`
      3. Get statuses, return `!statuses.is_empty()`
    - `pub fn git_status_summary(project_path: &str) -> Result<StatusSummary, GitError>`:
      1. Open repo, get statuses
      2. Count modified, added, deleted, untracked
      3. Return `StatusSummary` struct (derive Serialize): modified (usize), added (usize), deleted (usize), untracked (usize), is_clean (bool)

    **git/mod.rs** - Re-exports:
    ```rust
    pub mod branches;
    pub mod error;
    pub mod status;
    ```
