Define two new structs in `diff.rs`:

```rust
#[derive(Debug, Clone, serde::Serialize)]
pub struct DiffSummaryData {
    pub files_changed: usize,
    pub insertions: usize,
    pub deletions: usize,
    pub files: Vec<DiffFileEntry>,
    pub commits: Vec<super::merge::CommitInfo>,  // Reuse CommitInfo from merge.rs
}

#[derive(Debug, Clone, serde::Serialize)]
pub struct DiffFileEntry {
    pub path: String,
    pub insertions: usize,
    pub deletions: usize,
}
```

Implement `pub fn get_branch_diff_summary(project_path: &str, source_branch: &str, merge_target: &str) -> Result<DiffSummaryData, GitError>`:

1. Open repo with `Repository::open(project_path)?`
2. Find source and target branches with `repo.find_branch(name, BranchType::Local)?`
3. Get trees: `branch.get().peel_to_tree()?`
4. Compute diff: `repo.diff_tree_to_tree(Some(&target_tree), Some(&source_tree), None)?`
5. Get aggregate stats: `diff.stats()?.files_changed()`, `.insertions()`, `.deletions()`
6. Compute per-file stats using `diff.foreach()` with the line callback. Use a `HashMap<String, (usize, usize)>` to accumulate per-file insertions/deletions. In the line callback, match `line.origin()` for '+' (insertion) and '-' (deletion). Get file path from `delta.new_file().path()` falling back to `delta.old_file().path()`.
7. For commit list, reuse the revwalk pattern from `merge.rs::merge_preview`: find merge base with `repo.merge_base(source_oid, target_oid)?`, then walk from source tip to merge base using `Revwalk` with `Sort::TOPOLOGICAL | Sort::TIME`, pushing source oid, hiding merge base. Build `CommitInfo` for each commit.
8. Sort files alphabetically by path before returning.
9. Return `DiffSummaryData` with all fields populated.

In `src-tauri/src/git/mod.rs`, add `pub mod diff;` to the module declarations.

In `src-tauri/src/commands/git_commands.rs`, add a new Tauri command:

```rust
}
```

In `src-tauri/src/lib.rs`, register the new command in the `invoke_handler` macro, after the existing git commands:


Handle edge cases:
- If source and target point to same commit (0 commits ahead), return empty DiffSummaryData with 0s and empty vecs.
- If merge base cannot be found, return an error via `GitError::Other("No common ancestor found")`.

**A. Exit code capture in Rust (src-tauri/src/terminal/pty.rs + mod.rs):**

Modify `TerminalSession` in `mod.rs` to share the child process with the reader thread. Change `child` field from `Box<dyn Child + Send>` to `Arc<Mutex<Box<dyn Child + Send>>>`. Add the necessary imports: `use std::sync::{Arc, Mutex};`.

Update `pty.rs`:
1. After creating the `child` from `pair.slave.spawn_command(cmd)`, wrap it: `let child = Arc::new(Mutex::new(child));`
2. Clone for the reader thread: `let child_for_reader = Arc::clone(&child);`
3. In the reader thread, when EOF is detected (`Ok(0)`), call `child_for_reader.lock().unwrap().wait()` to get the exit status.
4. `portable_pty::ExitStatus` only provides `success()` (bool). Map it: if `success()` then `Some(0)`, else `Some(1)`. If `wait()` fails, send `None`.
5. Send the real exit code: `on_event.send(TerminalEvent::Exit { code })`.
6. In the `TerminalSession` struct construction at the bottom of `spawn_pty`, use the `Arc<Mutex<>>` wrapped child.

Also update the `Drop` impl and `TerminalManager::remove` -- anywhere that accesses `session.child` must now lock the mutex. The `terminal_kill` command in `terminal/commands.rs` may call `child.kill()` -- check and update that call to `child.lock().unwrap().kill()`.

```typescript
export function fireExitToast(terminalId: string, branchName: string, exitCode: number | null) {
  const isClean = exitCode === null || exitCode === 0;
  dismissOldestIfAtCapacity();

  const action = {
    label: 'View Session',
    onClick: () => useTerminalStore.getState().focusSession(terminalId),
  };
  const onDismiss = (t: { id: string | number }) => untrackToast(t.id);
  const onAutoClose = (t: { id: string | number }) => untrackToast(t.id);

  if (isClean) {
    const id = toast(`${branchName} exited`, {
      description: 'Session completed successfully',
      duration: 5000,
      action,
      onDismiss,
      onAutoClose,
    });
    trackToast(id, false);
  } else {
    const id = toast.error(`${branchName} crashed`, {
      description: `Exited with code ${exitCode}`,
      duration: Infinity,
      action,
      onDismiss,
      onAutoClose,
    });
    trackToast(id, true);
  }
}
```
