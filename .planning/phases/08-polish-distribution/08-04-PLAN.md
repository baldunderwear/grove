**# Grove** ("Manage your trees.")

One-line description: A lightweight Windows desktop app for managing git worktrees and Claude Code sessions.

**## Features**
List all major features with brief descriptions:
- Worktree Dashboard: See all branches with status (ahead/behind, dirty/clean, active sessions)
- Session Launch: One-click Claude Code launch per worktree, or create new worktree + launch
- Smart Merge: Preview, execute, auto-resolve build file conflicts, bump build numbers, rename changelogs
- System Tray: Background operation, quick-launch worktrees from tray, notifications for merge-ready branches
- Auto-fetch: Configurable background fetch from remotes
- Keyboard Shortcuts: Ctrl+R refresh, Ctrl+N new worktree, Ctrl+, settings
- Auto-Update: Automatic update checking via GitHub Releases

**## Installation**
1. Download the latest installer from [GitHub Releases](https://github.com/baldunderwear/grove/releases)
2. Run the NSIS installer (.exe) -- it will download WebView2 if needed
3. Grove starts in the system tray

**## Quick Start**
1. Click the tray icon to open the dashboard
2. Click "Add Project" and select a git repository
3. Configure branch prefix (e.g., `wt/`) and merge target (e.g., `main`)
4. See all worktree branches, launch Claude Code sessions, merge when ready

**## Configuration**
- **Per-project:** Merge target branch, branch prefix, build file patterns, changelog settings
- **Global:** Refresh interval, notification preferences, auto-start with Windows, auto-fetch interval
- Config stored in `%APPDATA%/grove/config.json`
- Export/import configuration from Settings

**## Keyboard Shortcuts**
Table of shortcuts: Ctrl+R/F5 (refresh), Ctrl+N (new worktree), Ctrl+, (settings), Escape (close dialog)

**## Development**
```bash
# Prerequisites: Node.js, Rust, git

cd src-ui && npm install

cargo tauri dev

# Build release
cargo tauri build

# TypeScript check
cd src-ui && npm run typecheck
```

**## Architecture**
Brief overview:
- `src-tauri/` -- Rust backend (Tauri commands, git2 operations, file watching, process management)
- `src-ui/` -- React 19 + TypeScript + Tailwind CSS frontend
- State: Zustand stores for branches, config, sessions, merge
- Git: git2 crate for local operations, git CLI for fetch (SSH agent compat)
