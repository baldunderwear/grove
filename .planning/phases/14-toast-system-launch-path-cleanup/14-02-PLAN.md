<objective>
Remove the entire external launch path (wt.exe/cmd.exe, PID-based session detection, session-store) and rewire all consumers to use the embedded terminal via terminal-store.

Purpose: SessionManager + terminal-store is now the sole path for launching Claude Code. The external launch infrastructure from v1.0 is dead code that confuses the codebase and wastes binary size (sysinfo crate). Removing it simplifies the architecture and prevents users from accidentally launching sessions outside the embedded terminal.

Output: Clean codebase with no external launch path, all consumers rewired to terminal-store, session-store deleted, Rust process/ module deleted, sysinfo crate removed.
</objective>
