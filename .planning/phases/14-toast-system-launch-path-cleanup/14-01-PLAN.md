3. Expand `src-ui/src/lib/alerts.ts` with toast functions. Keep ALL existing functions (`playAttentionChime`, `requestWindowAttention`, `fireWaitingAlert`). Add new exports:

   - `fireSessionToast(terminalId: string, branchName: string, state: string)`:
     - Per CONTEXT.md decisions: only fire for "waiting", "idle", "error" states. Do NOT fire for "working" or null.
     - For "error" state: use `toast.error()` with `duration: Infinity`
     - For "waiting"/"idle": use `toast()` with `duration: 5000`
     - All session toasts include `action: { label: 'View Session', onClick: () => useTerminalStore.getState().focusSession(terminalId) }`
     - Toast title format per UI-SPEC: "{branchName} needs input" (waiting), "{branchName} error" (error), "{branchName} idle" (idle)
     - Toast description per UI-SPEC: "Session is waiting for your response" (waiting), "Session encountered an error" (error), "Session has gone idle" (idle)

   - `fireErrorToast(title: string, description: string)`:
     - System error toast with `duration: Infinity`
     - No session action button (this is for non-session errors)

   - `fireSessionAlert(terminalId: string, branchName: string, state: string)`:
     - This is the main entry point called from terminal-store
     - Calls `fireSessionToast()` always (in-app toast per CONTEXT.md: "toasts fire always")
     - For "waiting" state only: also calls `playAttentionChime()` and checks window focus
     - Only fires OS notification (`requestWindowAttention`) when window is NOT focused (per CONTEXT.md: "OS notifications: fire only when app is unfocused")
     - This replaces the old `fireWaitingAlert` as the primary alert function for state changes

   - Add a module-level `activeToasts` array tracking `{ id, isError, timestamp }` for priority management. Before firing a new toast, if `activeToasts.length >= 3`, dismiss the oldest non-error entry using `toast.dismiss(id)`. When a toast is dismissed (via Sonner's `onDismiss` or `onAutoClose` callbacks), remove it from the array.

   - Import `toast` from `sonner` and `useTerminalStore` from `@/stores/terminal-store`
   - Import `getCurrentWindow` from `@tauri-apps/api/window` (already imported)
