# Requirements Archive — v1.0: Worktree Manager for Claude Code

**Archived:** 2026-03-28
**Status:** All requirements satisfied

## Functional Requirements

### FR-01: Project Registry — ALL COMPLETE
- [x] FR-01.1: User can add a project by selecting a git repository path
- [x] FR-01.2: User can remove a project from the registry
- [x] FR-01.3: Each project stores its own configuration
- [x] FR-01.4: Projects without build numbers work with plain merge
- [x] FR-01.5: Configuration persists across app restarts
- [x] FR-01.6: App detects if registered project path no longer exists

### FR-02: Worktree Dashboard — ALL COMPLETE
- [x] FR-02.1: Display all worktree branches for selected project
- [x] FR-02.2: Show per-branch: name, ahead/behind, last commit, date
- [x] FR-02.3: Show per-branch: dirty/clean status
- [x] FR-02.4: Show per-branch: active Claude Code session badge
- [x] FR-02.5: Sort branches by activity/name/commits
- [x] FR-02.6: Visual indicators: merge-ready, stale, active session
- [x] FR-02.7: Auto-refresh on configurable interval
- [x] FR-02.8: Manual refresh button

### FR-03: Session Launch — ALL COMPLETE
- [x] FR-03.1: One-click launch of Claude Code in worktree
- [x] FR-03.2: Launch with configurable flags (backend supports, UI deferred)
- [x] FR-03.3: Create new worktree with custom name and launch
- [x] FR-03.4: Track active Claude Code processes
- [x] FR-03.5: Open worktree in Explorer or VS Code

### FR-04: Merge Workflow — ALL COMPLETE
- [x] FR-04.1: Preview merge with commits, changelogs, build number
- [x] FR-04.2: Execute atomic merge with auto-resolve and build bump
- [x] FR-04.3: Rename changelog fragments
- [x] FR-04.4: Handle legacy numbered changelogs
- [x] FR-04.5: Detect and surface unexpected conflicts
- [x] FR-04.6: Confirmation dialog before merge
- [x] FR-04.7: Post-merge summary
- [x] FR-04.8: Merge is local only

### FR-05: System Tray — ALL COMPLETE
- [x] FR-05.1: System tray icon with context menu
- [x] FR-05.2: Tray menu with quick-launch worktrees
- [x] FR-05.3: Tray notifications for merge-ready, stale, completed
- [x] FR-05.4: Left-click opens, right-click menus
- [x] FR-05.5: Start minimized to tray (configurable)
- [x] FR-05.6: Start with Windows option

### FR-06: Git Status Monitoring — ALL COMPLETE
- [x] FR-06.1: Watch project directories for git changes
- [x] FR-06.2: Detect new worktree branches
- [x] FR-06.3: Detect deleted worktree branches

### FR-07: Settings — ALL COMPLETE
- [x] FR-07.1: Global settings
