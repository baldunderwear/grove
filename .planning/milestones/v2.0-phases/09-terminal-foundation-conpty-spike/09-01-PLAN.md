         ```rust
         }
         ```

    2. Create `src-tauri/src/terminal/pty.rs`:
       - `pub fn spawn_pty(working_dir: &str, cols: u16, rows: u16, on_event: Channel<TerminalEvent>) -> Result<(String, TerminalSession), String>`
       - Use `portable_pty::native_pty_system()` to open PTY with PtySize

       - Set env vars: `TERM=xterm-256color`, `COLORTERM=truecolor`
       - Spawn child via `pair.slave.spawn_command(cmd)`
       - Clone reader via `pair.master.try_clone_reader()`
       - Spawn dedicated `std::thread::spawn` for blocking read loop (per research Pattern 2):
         - 4096-byte buffer
         - `String::from_utf8_lossy` conversion
         - Send `TerminalEvent::Data` on successful read
         - Send `TerminalEvent::Exit` on EOF (read returns 0)
         - Send `TerminalEvent::Error` on error
         - Break loop if channel.send() returns Err (frontend disconnected)
       - Return terminal ID (uuid::Uuid::new_v4) and TerminalSession with writer + child
