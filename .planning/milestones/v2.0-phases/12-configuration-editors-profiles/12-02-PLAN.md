2. Create `src-ui/src/hooks/useFileEditor.ts`:
   - Export `useFileEditor(filePath: string | null)` hook returning `{ content, setContent, loading, error, dirty, save, reload }`.
   - On mount (and when filePath changes): call `invoke<string>('read_text_file', { path: filePath })` to load content. Track `originalContent` for dirty detection.
   - `dirty` = `content !== originalContent`.
   - `save()`: call `invoke('write_text_file', { path: filePath, content })`, then update originalContent. Return success/error.
   - `reload()`: re-fetch from disk, reset dirty state.
   - Track load start time with `performance.now()`, log to console if load exceeds 100ms (NFR-09 monitoring).
   - Return `loading: true` during initial fetch, `error: string | null` on failure.
