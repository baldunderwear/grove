  <action>
1. Create `src-ui/src/stores/branch-store.ts` -- a dedicated Zustand store for branch data, separate from config store to prevent unnecessary re-renders (branch data refreshes every 30s, config rarely changes):

   State fields:
   - `branches: BranchInfo[]` (default: [])
   - `loading: boolean` (default: false) -- true during initial fetch
   - `refreshing: boolean` (default: false) -- true during manual refresh (shows spinner)
   - `error: string | null` (default: null)
   - `sortMode: SortMode` (default: 'activity')
   - `lastRefreshed: number | null` (default: null) -- Date.now() timestamp of last successful fetch

   Actions:
   - `fetchBranches(path: string, prefix: string, target: string): Promise<void>` -- sets loading=true, calls `invoke<BranchInfo[]>('list_branches', { projectPath: path, branchPrefix: prefix, mergeTarget: target })`, sets branches + loading=false + lastRefreshed=Date.now(). On error: sets error string, loading=false. IMPORTANT: Use camelCase for invoke args (Tauri 2 auto-converts to snake_case for Rust).
   - `silentRefresh(path: string, prefix: string, target: string): Promise<void>` -- same as fetchBranches but does NOT set loading=true (no loading overlay). Only updates branches and lastRefreshed on success. Silently ignores errors (keeps existing data).
   - `manualRefresh(path: string, prefix: string, target: string): Promise<void>` -- sets refreshing=true, calls invoke, sets branches + refreshing=false + lastRefreshed. On error: sets error, refreshing=false.
   - `setSortMode(mode: SortMode): void` -- just sets sortMode, no data mutation (sorting is computed in component).
   - `clear(): void` -- resets branches to [], error to null, loading/refreshing to false, lastRefreshed to null. Called when switching projects to avoid showing stale data.

   Race condition protection: Include a `_fetchId` counter (not in state, just a module-level `let fetchCounter = 0`). In fetchBranches and manualRefresh, increment counter at start, check if counter still matches after await. If not, discard the result (another fetch superseded it).
