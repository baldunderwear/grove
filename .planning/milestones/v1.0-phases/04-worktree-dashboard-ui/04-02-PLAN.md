1. Create `src-ui/src/pages/Dashboard.tsx` -- the main dashboard page component:

   This component orchestrates data fetching, auto-refresh, event listening, and wires child components.

   Read from config store: `selectedProjectId`, `config` (to get the ProjectConfig for the selected project: path, branch_prefix, merge_target, name), `showProjectConfig` action.
   Read from branch store: `branches`, `loading`, `refreshing`, `error`, `sortMode`, `lastRefreshed`, `fetchBranches`, `silentRefresh`, `manualRefresh`, `setSortMode`, `clear`.

   Derive the selected project: `const project = config?.projects.find(p => p.id === selectedProjectId)`. If no project found, return null.

   **Effect 1: Initial fetch + project switch** (useEffect):
   - Deps: [project?.id, project?.path, project?.branch_prefix, project?.merge_target]
   - On mount or project change: call `clear()` then `fetchBranches(path, prefix, target)`.
   - Return cleanup: call `clear()`.

   **Effect 2: Auto-refresh timer** (useEffect):
   - Read `settings.refresh_interval` from config store (it stores seconds).
   - `setInterval` that calls `silentRefresh(path, prefix, target)` every `refresh_interval * 1000` ms.
   - Deps: [project?.path, project?.branch_prefix, project?.merge_target, settings?.refresh_interval]
   - Return cleanup: clearInterval.

   **Effect 3: File watcher event listener** (useEffect):
   - Import `listen` from `@tauri-apps/api/event`.
   - Listen to `'git-changed'` event. On event, check `event.payload.project_path` matches current project path (normalize backslashes for comparison). If match, call `silentRefresh(path, prefix, target)`.
   - Use the cancellation pattern from research (let cancelled = false, check in handler).
   - Deps: [project?.path, project?.branch_prefix, project?.merge_target]
   - Return cleanup: cancelled = true, call unlistenPromise.then(fn => fn()).

   **Effect 4: Window focus refresh** (useEffect):
   - Import `TauriEvent` from `@tauri-apps/api/event`.
   - Listen to `TauriEvent.WINDOW_FOCUS`. On focus, check if `lastRefreshed` is null or `Date.now() - lastRefreshed > 10_000`. If so, call `silentRefresh(path, prefix, target)`.
   - Use `useBranchStore.getState().lastRefreshed` to read current value (avoid stale closure).
   - Deps: [project?.path, project?.branch_prefix, project?.merge_target]
   - Return cleanup.

   **Render:**
   - Container: `div className="flex flex-col h-full px-8 pt-6"`
   - If error and not loading: show error state -- text-gray-400 centered message with "Failed to load branches. Check that the project path is accessible." and a "Try again" Button that calls fetchBranches.
   - DashboardHeader: pass projectName, branchCount (branches.length), lastRefreshed, refreshing, sortMode, onSortChange, onRefresh (calls manualRefresh), onShowConfig.
   - Gap: `mt-6` between header and content.
   - If loading: BranchTable with loading=true (shows skeletons).
   - Else if branches.length === 0: BranchEmptyState with prefix={project.branch_prefix}.
   - Else: BranchTable with branches, sortMode, loading=false, refreshing.
