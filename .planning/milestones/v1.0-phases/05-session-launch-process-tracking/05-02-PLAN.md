- `openInVscode`: invoke `open_in_vscode` with `{ worktree_path }`
- `openInExplorer`: invoke `open_in_explorer` with `{ worktree_path }`
- `clear`: reset activeSessions to `{}`

**Key:** The store does NOT own the polling interval. The Dashboard page will call `fetchSessions` on the same timer as branch refresh (per user decision: "polling interval matches dashboard refresh interval").
