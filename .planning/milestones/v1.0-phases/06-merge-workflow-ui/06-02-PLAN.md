    Create `src-ui/src/components/MergeDialog.tsx`:

    **Props interface:**
    ```
    interface MergeDialogProps {
      open: boolean;
      onOpenChange: (open: boolean) => void;
      branch: BranchInfo | null;
      project: ProjectConfig;
      onComplete: () => void;
    }
    ```

    **Component structure:**
    - Use `useMergeStore` for `preview, result, loading, error, step, fetchPreview, executeMerge, clearOperation`
    - On open (useEffect when open && branch): call `fetchPreview(project.path, branch.name, project.merge_target, project.build_files, project.changelog)`
    - On close: call `clearOperation()`
    - Guard `onOpenChange`: only allow close when step is NOT 'executing'. Wrap: `const handleOpenChange = (v: boolean) => { if (!v && step === 'executing') return; if (!v) clearOperation(); onOpenChange(v); }`

    **Dialog shell:**
    - `<DialogContent className="sm:max-w-lg">` with `onPointerDownOutside={(e) => { if (step === 'executing') e.preventDefault(); }}` and `onEscapeKeyDown={(e) => { if (step === 'executing') e.preventDefault(); }}` and `showCloseButton={step !== 'executing'}`
    - Render step-specific content via conditional blocks (not separate components -- keep it in one file for simplicity)

    **Step: loading (step === 'idle' && loading === true, or step === 'preview' && loading):**
    - DialogTitle: "Merge Preview"
    - Spinner/loading indicator: a simple `<div className="flex items-center justify-center py-8"><div className="animate-spin ...">` or use Loader2 from lucide-react
    - Text: "Loading preview..."

    **Step: preview (step === 'preview' && preview !== null):**
    - DialogTitle: `Merge ${preview.source_branch}`
    - DialogDescription: `into ${preview.target_branch}`
    - Content sections:
      1. **Commits** — List preview.commits_to_merge with oid (first 7 chars), message (truncated), author. Use ScrollArea if > 5 commits. Show count: "{N} commits to merge"
      2. **Changelog Fragments** — If changelog_fragments.length > 0: list each with name, "(legacy)" badge if is_legacy. If none: omit section.
      3. **Build Number** — If current_build !== null: "Build {current_build} -> {next_build}". If null: omit section.
      4. **Conflicts Warning** — If has_conflicts: render an amber/red warning box: "This merge has conflicts in non-build files. Auto-resolution is not possible." Show warning icon.
    - Footer:
      - Cancel button (variant="ghost")
      - If has_conflicts: "Abort" button only (no proceed) -- per decision "User can abort merge if unexpected conflicts found" (FR-04.5)
      - If no conflicts: "Continue" button (variant="default") -> sets step to 'confirm' via `useMergeStore.setState({ step: 'confirm' })`

    **Step: confirm (step === 'confirm'):**
    - DialogTitle: "Confirm Merge"
    - Content:
      - Summary line: "Merge {source_branch} into {target_branch}"
      - If build number: "Build number will bump from {current_build} to {next_build}"
      - If changelog fragments: "{N} changelog fragment(s) will be renamed"
      - "{N} commits will be merged"
      - Notice in muted text: "This is a local merge only -- changes will not be pushed to remote." (per FR-04.8)
    - Footer:
      - "Back" button (variant="ghost") -> sets step back to 'preview'
      - "Merge Branch" button (primary, emerald accent: `className="bg-emerald-600 hover:bg-emerald-700 text-white"`) -> calls `executeMerge(project.path, branch.name, project.merge_target, project.build_files, project.changelog)` (FR-04.6)

    **Step: executing (step === 'executing'):**
    - DialogTitle: "Merging..."
    - Simple spinner + "Merging {source_branch} into {target_branch}..." text
    - No buttons (dialog cannot be closed -- guarded above)

    **Step: summary (step === 'summary' && result !== null):**
    - DialogTitle: "Merge Complete" with a check icon (CheckCircle2 from lucide-react, text-emerald-500)
    - Content (per FR-04.7):
      - "Successfully merged {result.commits_merged} commits"
      - If result.new_build: "Build number: {result.new_build}" in emerald badge
      - If result.changelog_renames.length > 0: list each rename as "{from} -> {to}"
      - If result.warnings.length > 0: amber warning section listing each warning
    - Footer:
      - "Done" button (variant="default") -> calls onComplete(), then onOpenChange(false)

    **Step: error (step === 'error'):**
    - DialogTitle: "Merge Failed"
    - Show error string in text-destructive
    - Footer: "Close" button -> onOpenChange(false)

    **Icons:** Import GitMerge, Loader2, CheckCircle2, AlertTriangle from lucide-react.
